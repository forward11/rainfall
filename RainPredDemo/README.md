# 六盘山神经网络评估
1. 全连接神经网络
2. train:2008-2017  val:2018-2019  test:2020
3. 使用六盘山气象站的数据进行预测，历史回归模型
4. 对风速风向进行了修正，结果仍然不明显
5. r_squre的引入，发现模型对2019年的预测结果极差，需要修改
